Parameters

In the Getting Started tutorial, we automated a simple task but we noticed there was still a lot of typing.

RobotWare has parameters which is a way of referring to the same piece of text. For example, in the Getting Started tutorial, the directory we created (c:\temp\RobotWare) was used in several places. If we changed our the directory name, we would then have to change it everywhere it is used. Parameters allow you to change it in one place and have it reflected everywhere it is used.

Let's automate the same simple task to show how to use RobotWare with parameters:

  1. delete an existing directory
  2. create the directory again
  3. copy an existing image file to the new directory
  4. resize the image to 400x400 pixels
  5. save the resized image to a new name
  6. play a system sound to let us know it is finished

Doing this in Windows is tedious but is easily done in RobotWare:

  1. start RobotWare
  2. click New... button
  3. Editor dialog is displayed

  4. click Parameters... button
  5. Parameters Editor dialog is displayed

  6. click Add... button
  7. Add Parameter dialog is displayed
  8. in Name box, type ImageSourceDir
  9. in Value box, type c:\temp\RobotWare
  10. this will be the directory we will delete and then recreate
  11. click OK button

  12. click Add... button again and create parameters for:

    13. NameValueComment
    ImageSourceDirc:\temp\RobotWaredirectory we will delete and then recreate
    OriginalImageName[use browse button to select file]file name of image to be resized
    OriginalImageSourceDir[use browse button to select directory]directory of image to be resized
    ResizedImageName[enter a new image file name]file name of resized image

  13. we should now be in the Parameters Editor dialog, so click OK button to go back to the Editor dialog

  14. click Add... button
  15. Add Step dialog is displayed
  16. in Search box, type delete
  17. double click on Delete a directory step
  18. Add Step dialog is closed and Delete a directory step is added to Editor
  19. in Source box, right click
  20. from the popup menu which appears, select ImageSourceDir
  21. in the box above, notice how the summary of the step says Delete c:\temp\RobotWare and any subdirectories and files

  22. click Add... button
  23. Add Step dialog is displayed
  24. in Search box, type create
  25. double click on Create a directory step
  26. Add Step dialog is closed and Create a directory step is added to Editor
  27. in Source box, right click and select ImageSourceDir
  28. in the box above, notice how the summary of the step says Create c:\temp\RobotWare

  29. click Add... button
  30. in Search box, type copy
  31. double click Copy a file step
  32. Add Step dialog is closed and Copy a file step is added to Editor
  33. in Source box, right click and select OriginalImageSourceDir
  34. in Source box, type \
  35. in Source box, right click and select OriginalImageName
  36. in Destination box, right click and select ImageSourceDir
  37. in Destination box, type \
  38. in Destination box, right click and select OriginalImageName
  39. in the box above, notice how the summary of the step has updated with the values of the parameters

  40. click Add... button
  41. in Search box, type image
  42. double click Resize an image by pixels step
  43. in Source box, right click and select ImageSourceDir
  44. in Source box, type \
  45. in Source box, right click and select OriginalImageName
  46. in Destination box, right click and select ImageSourceDir
  47. in Destination box, type \
  48. in Destination box, right click and select ResizedImageName
  49. in the box above, notice how the summary of the step has updated with the values of the parameters
  50. in Pixels Wide and Pixels High boxes, type 400

  51. click Add... button
  52. in Search box, type sound
  53. double click Play a Windows system sound step
  54. in Play drop down list, select Exclamation

  55. click Save button
  56. Save dialog is displayed
  57. in box, enter RobotWare Parameters Tutorial and click Add... button

  58. click Run button
  59. each step is performed and a sound is played at the end
What we now have is a Specification which we can run again at a later time.

More importantly, we have a Specification which uses parameters to make it easily modifiable if, for example, we would like to resize a different image file. In this case, we only need to change one parameters in one place to make the change.

Once you have mastered parameters, have a look at Batch Operations which gives you even more power and flexibility.
