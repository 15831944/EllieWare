RobotWare

RobotWare is a simple, 'code free' environment for automating tasks. It allows non-programmers to easily develop macros to do repetitive tasks.

At the lowest level is a step which is a small piece of functionality eg:

  • deleting a file on disk
  • copying a file
  • creating a zip archive
A variety of steps are then chained together to form a specification eg:
  • delete an image file
  • copy a new image file
  • resize image file
  • create a zip archive of image file
  • delete zip archive on FTP server
  • upload new zip archive to FTP server
  • play a sound
RobotWare will then execute each of these steps in the specification in sequence to automate a task.
