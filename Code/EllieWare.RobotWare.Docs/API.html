Supported Interfaces

There are only two interfaces you need to implement to create your own RobotWare plugin:

  • IFactory

    • A lightweight interface which tells RobotWare only enough information for it to display an option for the plugin to the user
  • IRunnable or IMutableRunnable

    • Most of the work will be done in this interface. Implement IMutableRunnable if your plugin has settings which the user can change through the plugin user interface.

For more information, see the EllieWare.Interfaces.Docs.chm help file.
