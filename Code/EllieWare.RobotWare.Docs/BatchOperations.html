Batch Operations

Batch operations combines the ideas of parameters and running other specifications. It allows you to run a series of specifications, with the parameters coming from:

  • files in a directory
  • lines in a text file

For example, this can be used to perform the same series of operations on all files in a directory:

  • resize image file
  • convert image file to grayscale
  • convert image file to a different format
  • move converted image file

To add a batch operation, add a step to your specification and select Batch from the Category dropdown. There should only be one step listed.

The configuration user interface for batch operation allows you to:

  • add/remove and reorder other specifications
  • select a directory or file for the batch parameters

Once you have added other specifications, when you edit one of the specifications, additional context menu items are available:

  • Batch Parameter

    • [Directory] name of file including extension eg MyImage.png
    [File] line of text from file eg c:\Documents\MyDocuments\MyImage.png

  • Batch Parameter [Directory]

    • [Directory] fully qualified path to directory not including trailing backslash eg c:\Documents\MyDocuments

  • Batch Parameter [FileName]

    • [Directory] name of file not including extension eg MyImage
