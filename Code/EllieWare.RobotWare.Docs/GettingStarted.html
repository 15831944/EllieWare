Getting Started

Let's automate a simple task to show how to use RobotWare:
  1. create a new directory
  2. copy an existing image file to the new directory
  3. resize the image to 400x400 pixels
  4. save the resized image to a new name
  5. play a system sound to let us know it is finished

Doing this in Windows is tedious but is easily done in RobotWare:

  1. start RobotWare
  2. click New... button
  3. Editor dialog is displayed

  4. click Add... button
  5. Add Step dialog is displayed
  6. in Search box, type create
  7. double click on Create a directory step
  8. Add Step dialog is closed and Create a directory step is added to Editor
  9. in Source box, type c:\temp\RobotWare

  10. click Add... button
  11. in Search box, type copy
  12. double click Copy a file step
  13. Add Step dialog is closed and Copy a file step is added to Editor
  14. in Source box, use the button marked with ... to browse for an image file
  15. in Destination box, type c:\temp\RobotWare\ and the name of the image file

  16. click Add... button
  17. in Search box, type image
  18. double click Resize an image by pixels step
  19. in Source box, type c:\temp\RobotWare\ and the name of the image file
  20. in Destination box, type c:\temp\RobotWare\ and a different image file name
  21. in Pixels Wide and Pixels High boxes, type 400

  22. click Add... button
  23. in Search box, type sound
  24. double click Play a Windows system sound step
  25. in Play drop down list, select Exclamation

  26. click Save button
  27. Save dialog is displayed
  28. in box, enter RobotWare Tutorial and click Add... button

  29. click Run button
  30. each step is performed and a sound is played at the end
What we now have is a Specification which we can run again at a later time.

You may have noticed there was a lot of typing the same directory and file paths. If we changed the original image, we then had to change the name in subsequent steps. This is both tedious and error prone.

Luckily RobotWare allows us to setup and use Parameters for exactly this situation.
