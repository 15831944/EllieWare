Debugging

Most times, a RobotWare specification will run without problems. Occasionally, things will not work as expected.

To diagnose where your specification is going wrong, there are the following features:

  • Single step in the Editor

    • Allows you to execute the specification one step at a time. You should then see which step fails. This is the recommended method for debugging a specification.
  • Debug statements

    • Record information to the log window. These can be put in a specification at various places to monitor progress. This feature is available as a builtin RobotWare feature under the Debugging category.
